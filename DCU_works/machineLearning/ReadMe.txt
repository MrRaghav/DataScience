Video memorability has various applications.

In this project, features extracted by a different team have been utlized to predict the short- and long-term memorability of the videos.

Features used: 
--------------
HMP
C3D
C3D with HMP
captions with count vectorizer
captions with tf-idf
captions with HMP
captions with C3D

Regression techniques used: 
---------------------------
Linear Regression
KNN
Decision tree
Random forest
VotingRegressor
AdaBoostRegressor
GradientBoostingRegressor
Neural Network 

After analyzing the spearman's score for all of them, the best regressor was used on the test set.
